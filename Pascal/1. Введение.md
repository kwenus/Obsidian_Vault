файл с суффиксом **.pas**
компилятор **fpc**

- hello.pas
```Pascal
program hello; // заголовок программы, идентификатор
begin // начало программы
	writeln('Hello, world') 
	// оператор вывода с переводом строки + строковый литерал в кавычках
end. // конец программы, точка обязательна!
```

#Pascal компиляция программы
```bash
fpc hello.pas
```

#Pascal возможные сбои компиляции:
- Error и Fatal - сообщения об ошибке, т.е. текст не соответствует правилам языка, поэтому его нельзя скомпилировать
- warning - предупреждение о том, что программа будет работать неправильно
- note - замечание, что какая-то часть программы непонятна компилятору, но работать будет. 

примеры ошибок:
```bash
hello.pas(3,12) Error: Identifier not found "writenl"
// компилятор не знает слово writenl находящееся в строке 3 в столбце 12

hello.pas(5) Fatal: Syntax error, "." expected but "end of file" found
// компилятор не нашел точку в конце программы и не может её скомпилировать
```

при удачной компиляции получаем:
hello.o - **объектный модуль**, который компилятор подает на вход компоновщику
hello - **исполняемый файл**, содержащий машинный код, соответствующий программе.

#Pascal запуск программы:
```bash
./hello
```

#Pascal разделение операторов
```Pascal
program hello;
begin
	writeln('Hello, world'); 
	write('Good night!') // оператор вывода без перевода строки
end.
```
